  - 建造者模式
  - 原型模式
- 行为型（7 种）

  - 装饰器模式
  - 代理模式
  - 外观模式
  - 桥接模式
  - 组合模式

  - 解释器模式

## 设计模式在哪里用到呢？

项目 => 面向对象（OO）=> 功能模块（设计模式+算法（数据结构）） => 框架（多种设计模式++算法（数据结构））=> 架构 => 服务器集群

使用设计模式，代码具有可扩展性，可读性，具有体系。
